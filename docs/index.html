parc – projector aspect ratio correction

This is a small piece of JavaScript code that allows you to adjust the scale and translation of your HTML page, in cases where it's being projected in an incorrect aspect ratio.

Have you ever been stuck using a projector that just didn't get the aspect ratio right? I have, and was extremely annoyed by it. As a response I created this project, which simply uses the transform CSS property to adjust your HTML page.

How it works

You simply need to include the JavaScript library in your HTML: 

	<script src="parc.js"></script>

(Protip: include it at the end of your HTML body, like you're supposed to do with jQuery and other libraries.)

At this point, assuming there are no JavaScript errors in the console, you're good to go. Here are the keys you can hit:

  • Press V to display a square with the intention of adjusting the page's scale. At this point you should be able to use your arrow keys to adjust the vertical & horizontal scale from anywhere between 0.5 and 1.0. When you're done, hit the V key again to turn off the square.
  • Press B to display a square with the intention of adjusting the page's offset (or translation). Again, you should be able to use your arrow keys to change the offset, and you can hide the square by hitting the B key again.
  • The code tries to do a guess as to what color (black or white) would show up properly on the screen, but if you can't see the square you can invert the color by hitting the X key.

Changing the default values

If your HTML page is already listening for keypresses, and you've already got assignments for the above keys, you can set up some default values for the keys. Just use the following code, changing the values to something more appropriate: 

    <script>
        window.parc = {
            aspectKey:    'V',
            translateKey: 'B',
            invertKey:    'X',
            strokeColor:  '#fff'
        };
    </script>
    <script src="parc.js"></script>

Make sure you define the parc object before the script is included, so that your default values are used by the included script.
